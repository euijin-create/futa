# 유타

원유가격 기반 일본행 항공권 발권 타이밍 참고 서비스입니다. 원유가격을 USD/KRW 환율로 원화로 바꿔 보여줍니다.

## 소개

이 앱은 항공권 가격이나 실제 유류할증료를 예측하지 않습니다. 오피넷 국제 원유가격 추이를 바탕으로, 일본행 항공권을 지금 살지 조금 더 기다릴지 참고할 수 있게 돕는 초간단 MVP입니다.

## 포함하지 않는 것

- 이 앱은 실제 항공권 가격 예측 서비스가 아닙니다.
- 이 앱은 실제 항공사 유류할증료 고시표를 사용하지 않습니다.
- 원유가격은 유류할증료의 간접 참고지표일 뿐입니다.
- 환율, 항공사 정책, 거리 구간, 발권일 기준 고시금액은 반영하지 않습니다.
- MVP이므로 일본행 항공권을 고민하는 사용자를 대상으로 한 참고 서비스입니다.

## 데이터

- 입력 파일: `data/oil_prices.csv`
- 환율 파일: `data/usd_krw.csv`
- 원유 파일 컬럼: `date`, `dubai`, `brent`, `wti`
- 환율 파일 컬럼: 날짜와 `달러(원)` 계열 컬럼
- `0.00` 값은 결측치로 처리합니다.

## 실행 방법

로컬 실행:

```bash
python app.py
```

Vercel 배포:

1. 이 GitHub 저장소를 Vercel 프로젝트로 import합니다.
2. Framework Preset은 `Flask` 또는 자동 감지를 사용합니다.
3. 배포하면 `app.py`의 Flask 앱을 기준으로 자동 배포됩니다.

## 화면 사용법

1. 일본 목적지를 하나 고릅니다.
2. 기준일은 최근 기간의 마지막 날로 생각하면 됩니다.
3. 비교 기간은 `30일`을 먼저 추천합니다.
4. 버튼을 누르면 최근 기간과 그 이전 같은 기간을 자동 비교합니다.

## 기능

- 이전 기간과 최근 기간의 평균 원유가격 비교
- 증감률 계산
- 일본 목적지 선택
- 기준일 1개와 비교 기간 1개만 선택하는 단순한 날짜 방식
- Dubai, Brent, WTI 선그래프
- 원화 기준 기간별 평균 막대그래프
- 간단한 추천 문구 표시

## 추천 로직

- `percent_change > 3`
  - 최근 원유가격이 상승세로 보고, 지금 발권을 검토하는 쪽을 참고합니다.
- `percent_change < -3`
  - 최근 원유가격이 하락세로 보고, 조금 기다려보는 선택지를 참고합니다.
- `-3 <= percent_change <= 3`
  - 원유가격 변동이 크지 않은 것으로 보고, 발권 시점 차이가 크지 않다고 참고합니다.

## 주의

이 서비스는 원유가격과 환율을 간접 지표로 사용합니다. 실제 항공권 구매 결정은 항공사 정책, 좌석 상황, 발권 시점 등 다른 요소도 함께 고려해야 합니다.

## Vercel 메모

Vercel의 Python 런타임은 Flask 같은 WSGI 앱을 지원하고, 저장소의 `app.py`에 있는 `app` 객체를 엔트리포인트로 읽을 수 있습니다. Git 저장소를 연결하면 브랜치 push마다 자동 배포가 가능합니다.
